# Mapa Interactivo Multicapas de Sismos en Cuauhtémoc

Este repositorio contiene un mapa interactivo en HTML que muestra la distribución de algunos elementos en la alcaldía Cuauhtémoc, Ciudad de México (Alcaldía, hospitales, vegetación, flora, fauna, etc).  
El mapa fue generado con **Python, Folium y GeoPandas**, integrando datos de instituciones gubernamentales.

---

## Contenido del repositorio
- `mapa_cuauhtemoc_multicapas.html` → Archivo principal con el mapa interactivo.
- `README.md` → Documento de descripción del proyecto.

---

## Instalación
No requiere instalación especial.  
Para visualizar el mapa:
1. Descarga el archivo `mapa_cuauhtemoc_multicapas.html`.
2. Ábrelo en cualquier navegador web moderno (Chrome, Edge, Firefox).

---

## Uso
El mapa permite:
- Visualizar la distribución de los elementos paisajisticos en Cuauhtémoc.
- Activar y desactivar capas para corroborar la presencia o ausencia de eventos.
- Consultar información básica de cada sismo mediante pop-ups.
